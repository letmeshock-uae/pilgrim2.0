
# DESIGN RULES

- Use 'Playfair Display' (Serif) for all H1/H2 headings.
- Use 'Inter' (Sans) for all chat bubbles and button labels.
- NO DEAD BUTTONS: Every interactive element must have a 'console.log' or state change.
- All cards must have a 1px border of `gray-100` for a "Museum" look.
- 3D Scene background must match the site background (#F9F9F9).
- Camera transitions in 3D must be smoothed using `dampE` or `damp3` from `maath`.
- Suggestion chips must have a hover/active scale of 0.95.
